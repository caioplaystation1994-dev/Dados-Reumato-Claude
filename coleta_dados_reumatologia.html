🅳 Exportar para Google Sheets

✓ Integração configurada! Clique em Exportar agora para enviar os dados para a planilha.
URL do Web App:
Esta integração usa o Google Apps Script — funciona diretamente do arquivo HTML, sem servidor. Siga os 5 passos abaixo uma única vez.
1
Criar a planilha

Acesse sheets.google.com e crie uma nova planilha em branco. Dê o nome que preferir (ex: "Dados Reumatologia").

2
Abrir o editor de script

Dentro da planilha, clique em Extensões → Apps Script. Uma nova aba abrirá com o editor de código.

3
Colar o código abaixo e salvar

Apague o código que já existe e cole este (depois pressione Ctrl+S para salvar):

function doPost(e) { try { var payload = JSON.parse(e.postData.contents); var values = payload.rows; // array de arrays (cabeçalho + dados) var ss = SpreadsheetApp.getActiveSpreadsheet(); // Aba principal: substitui tudo var sheet = ss.getSheetByName('Pacientes'); if (!sheet) sheet = ss.insertSheet('Pacientes'); else sheet.clearContents(); if (values && values.length > 0) { sheet.getRange(1, 1, values.length, values[0].length).setValues(values); var header = sheet.getRange(1, 1, 1, values[0].length); header.setBackground('#1a56a0'); header.setFontColor('#ffffff'); header.setFontWeight('bold'); sheet.setFrozenRows(1); sheet.autoResizeColumns(1, values[0].length); } // Aba de log: registra cada exportação var log = ss.getSheetByName('Log de Exportações'); if (!log) { log = ss.insertSheet('Log de Exportações'); log.appendRow(['Data/Hora', 'Pacientes exportados']); } log.appendRow([new Date(), values.length - 1]); return ContentService .createTextOutput(JSON.stringify({ ok: true, rows: values.length - 1 })) .setMimeType(ContentService.MimeType.JSON); } catch(err) { return ContentService .createTextOutput(JSON.stringify({ ok: false, error: err.message })) .setMimeType(ContentService.MimeType.JSON); } } // Permite testar se a URL está correta abrindo no navegador function doGet() { return ContentService .createTextOutput(JSON.stringify({ status: 'online', app: 'Dados Reumatologia' })) .setMimeType(ContentService.MimeType.JSON); }
4
Implantar como Web App

No editor, clique em Implantar → Nova implantação. Em "Tipo", selecione App da Web. Configure:

  • Executar como: Eu (sua conta Google)
  • Quem pode acessar: Qualquer pessoa

Clique em Implantar e autorize o acesso à planilha quando solicitado.
